Notes:
  • This container needs the same volume mount paths as your Plex server so it can write clearlogo.png directly next to your media.
  • Logos are saved as clearlogo.png in the movie's folder, or the show's root folder for TV.
  • After applying a logo, a Plex metadata refresh is triggered automatically for that item.
  • If no local clearlogo.png exists, the app asks Plex what clear logo it's actively showing (e.g. set via Plex Web, an agent, or another tool) and caches that result, so it won't query Plex again on future loads. Poster thumbnails and clear logo status are both cached locally — use "↻ Refresh Library" to re-check everything against Plex.
